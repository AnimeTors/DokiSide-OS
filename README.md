# **DokiSide OS — Cute Meets Uncanny**
*A cinematic Linux universe inspired by DDLC and MiSide.*

---

## 🌸🖤 **About**
DokiSide OS blends the pastel softness of *Doki Doki Literature Club* with the cozy‑uncanny atmosphere of *MiSide*.  
It features three complete themes, preinstalled games, creator tools, and a multiverse‑style theme switcher.

---

## 🎨 **Features**
- Three full desktop themes: **DDLC**, **MiSide**, **Hybrid**  
- Preinstalled **DDLC** and **MiSide Demo**  
- One‑click installers for **DDLC+** and **MiSide Full**  
- Preinstalled creator tools: **Ren’Py, Unity, Godot, Unreal**  
- Custom icons, sounds, wallpapers, and boot animations  
- VN‑style welcome screen  
- Multiverse theme switcher  
- Creator’s Corner app category
